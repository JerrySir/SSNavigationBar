# SSNavigationBar
对navigation Bar 进行的扩展，可以实现修改背景色、bar高度、左右button item的透明度、Y轴偏移
对于navigation Bar的偏移没有使用CGAffineTransformMakeTranslation，而是使用的自定义view作为遮罩进行的背景色改变、位置偏移操作。
SSNavigationBar.swift中有详细的使用说明，目前还在逐步增加功能
